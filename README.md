# cli-click-python

## Usage

You need to have installed python
> install python 3.11

Create The resources files for python Envioroment
> python -m venv .venv

Activate the Envioroment for python
- MacOS
> . .venv/bin/activate
- Windows 
> .venv\scripts\activate

### commands
Run
> python cli.py --help

You'll able to see the commands
- division
- producto
- resta
- suma

Replace --help with a command

> python cli.py suma
